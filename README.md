# DATA ANALYSIS PORTFOLIO

## Overview
This portfolio contains 5 data analysis projects across different domains:
- Sales Analysis
- Healthcare Analysis
- Sports Analytics
- Financial Analysis
- E-commerce Analytics

## Tools Used
- Python
- Pandas
- Matplotlib

## Projects

### 1. Sales Analysis
Analyzed sales trends, product performance, and customer behavior.

### 2. Healthcare Analysis
Analyzed patient data, treatment costs, and outcomes.

### 3. Sports Analytics
Analyzed player performance and team statistics.

### 4. Financial Analysis
Analyzed stock trends and price patterns.

### 5. E-commerce Analytics
Analyzed customer purchases and revenue.

## Conclusion
This portfolio demonstrates end-to-end data analysis including visualization and insights generation.
